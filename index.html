3D Ocean Simulation Project

The objective of this project is to create a 3D ocean simulation using mesh deformation, the Fast Fourier Transform, and 3D graphic shaders. This project will use many computational geometry techniques, such as ray tracing, and triangulation, as well as more advanced mathematical topics, such as space partitioning, frequency and time domain analysis, and oceanographic spectra, which when combined, aim to produce realistic and dynamic visualizations of ocean surfaces and subsurface interactions.

Realistic ocean simulations exist already, both in many modern games, and just ocean simulations by themselves. This project aims to accomplish similar things to past implementations, but with a web-deliverable. Many ocean simulations are full-fledged games, written in C++ or C# in Unreal Engine or Unity. This project aims to work entirely in the browser, using WASM and WebGL to render 3D environments in the browser. The implementation language of choice is C#.

Summary of the Final Presentation

Overview

  • 3D Real-time Ocean Wave Simulation
  • Large body of water with realistic waves using the Fast Fourier Transform
  • Realistic water appearance: reflections, refractions, and foams

Applications & Related Work

  • Used in video games like Sea of Thieves, Atlas, and Uncharted 4, as well as films
  • Typically simulated using the Fast Fourier Transform, referencing the Tessendorf paper

Basic Idea

The Fast Fourier Transform efficiently computes millions of waves by representing the wave height field at horizontal positions and time. The Phillips spectrum is used to simulate wind-generated waves.

Implementation

  • Simulation implemented in C# using OpenGL
  • FFT computations performed on the GPU with compute shaders
  • Water mesh optimized with GPU tessellation

Foam Implementation

Foam is created at wave collision points, blended with water color, and decays over time for realistic effects.

Reflections

Reflections are implemented using ray tracing. Rays are cast and reflected off the water surface, with the intersection points determining the reflection color.
